# HDB Resale Price Calculator

A beginner Python CLI project to explore Singapore HDB resale flat prices using real data from data.gov.sg.

## What it does
- Search resale prices by town, flat type, and year
- See average, min, and max prices for your search
- Save and view your search history
- Browse available towns and flat types

## How to run
```bash
python hdb_calculator.py
```
